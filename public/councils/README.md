# Council Member Images

Place the profile images of council members in this folder with the exact file names below.

1. **brijendra-kumar-mishra.jpg** - Dr. Brijendra Kumar Mishra (Convener)
2. **ashish-jha.jpg** - Mr. Ashish Jha (Secretary General)
3. **rajiv-ranjan.jpg** - Mr. Rajiv Ranjan (Deputy Secretary General)
4. **govind-kumar.jpg** - Mr. Govind Kumar (Deputy Secretary General)
5. **deepak-kumar-raj.jpg** - Dr. Deepak Kumar Raj (Asst. Secretary General)
6. **ruchika-kumar.jpg** - Ms. Ruchika Kumar (Finance Controller & Treasurer)

## Image Requirements:

- **Format**: JPG or JPEG
- **Size**: Recommended 400x400px to 800x800px (square aspect ratio)
- **File Size**: Keep under 200KB for optimal performance
- **Quality**: High quality professional headshots

## File Naming:

- Use **lowercase letters only**
- Use **hyphens** (not spaces or underscores) between words
- Include **.jpg** extension

## How to Add:

1. Download profile pictures from LinkedIn or other sources
2. Rename them to match the exact names above
3. Place them in this folder (`public/councils/`)
4. Refresh your browser - images should now appear in council cards
